Wixel-Servo-Lib by jpvlsmv

Library for controlling RC servos with a Pololu Wixel

This library supports controlling R/C PWM servos from a Pololu Wixel. It supports connecting the control servo to any of the GPIO pins. Keep in mind that servos often require a significant draw of power, usually at 5V. Since the Wixel is 3.3V, make sure that the power is properly regulated. This has been tested built the Wixel SDK compilers, it was developed in Eclipse (as provided by the TI CCS v4 studio)
